# TugasBesar-PraktikumGit-Kelompok[X]

Repositori ini dibuat untuk memenuhi Tugas Besar/Akhir mata kuliah Praktikum Version Control. 

## Tema Proyek
**Website Profil Kelompok**

## Struktur Anggota & Peran
- **Ketua Kelompok**: [Nama Anda] - Membuat repo & mengelola kolaborasi
- **Anggota 1**: [Nama Anggota 1] - Kontributor fitur / halaman
- **Anggota 2**: [Nama Anggota 2] - Kontributor fitur / halaman
- **Anggota 3**: [Nama Anggota 3] - Dokumentasi / konten

## Teknologi yang Digunakan
- HTML5
- Vanilla CSS (Modern Design)

## Aturan Kolaborasi (PENTING)
1.  **Dilarang push langsung ke branch main.**
2.  Setiap anggota wajib melakukan **Clone** repositori ke lokal.
3.  Setiap anggota wajib membuat branch sendiri dengan format: `kontribusi-[NIM]`.
4.  Lakukan perubahan di branch masing-masing.
5.  **Commit minimal 3 kali** dengan pesan yang bermakna.
6.  Push branch ke GitHub dan buat **Pull Request (PR)**.
7.  Ketua kelompok akan melakukan review dan **Merge**.
8.  Setelah merge, anggota lain wajib melakukan `git pull` untuk sinkronisasi.

## Struktur Folder
- `index.html`: Landing Page Utama.
- `css/`: Berisi file styling (global dan per anggota).
- `members/`: Berisi halaman profil masing-masing anggota.
- `assets/`: Berisi gambar dan aset lainnya.
