## Aturan Kolaborasi (PENTING)
1.  **Dilarang push langsung ke branch main.**
2.  Setiap anggota wajib melakukan **Clone** repositori ke lokal.
3.  Setiap anggota wajib membuat branch sendiri dengan format: `kontribusi-[NIM]`.
4.  Lakukan perubahan di branch masing-masing.
5.  **Commit minimal 3 kali** dengan pesan yang bermakna.
6.  Push branch ke GitHub dan buat **Pull Request (PR)**.
7.  Ketua kelompok akan melakukan review dan **Merge**.
8.  Setelah merge, anggota lain wajib melakukan `git pull` untuk sinkronisasi.
